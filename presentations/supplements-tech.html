Slide 04 · PWA

PWA implementation

  • Manifest: categories: ["lifestyle","health","productivity"]. Installable; install drives adherence reminders + reorder prompts.
  • Service worker: pre-cache app shell + fonts; never cache /api/recommend or /api/click.
  • Background Sync for partial-quiz submissions on flaky networks.
  • Push topics: adherence_morning, reorder_due, bundle_offer, winback. Quiet hours per timezone.
  • Strict CSP nonces on advertorial pages; encrypted IndexedDB resume of partial quizzes.
Slide 05 · Decision quiz

Goal-based decision quiz (≤ 90s)

  1. Primary goal (fat loss · appetite · energy · gut · sleep)
  2. Secondary goals (≤ 3, multi-select)
  3. Activity level + dietary pattern
  4. Risk filter (pregnancy, conditions, medications)
  5. Allergens (soy, dairy, gluten, shellfish, …)
  6. Lifestyle context (commute, work hours, sleep)
  7. Budget band
  8. Email · name · phone (optional, lifts retention)
  9. Marketing consent + age confirmation

Output: 3-product stack + bundle + comparison view → partner-brand checkout with HMAC-attributed redirect. Rule-based engine in V1; small XGBoost model in V2.

Slide 07 · Click flow

Edge attribution + signed redirect

// Cloudflare Worker: POST /api/click
validate(payload)
click_token = HMAC(user_id || product_id || ts)
persist click row (ip, ua, utm, click_token)
url = render(product.url_template, {
  click_token, sub_id: user_id, partner_id, ...
})
sendEvent("click_to_partner", click_token)
return 302 → url

// Inbound: POST /api/webhooks/partner/:slug
verify HMAC signature
match click_token → click row
insert partner_postbacks(event, amount, raw)
fan out:
  - Klaviyo flow trigger (order_paid → onboarding · sub_cancelled → winback)
  - Postscript trigger (reorder reminder · cancel-intent SMS)
  - server-side fanout (Meta CAPI · TikTok Events · GA4)

Slide 12 · Compliance

FTC-aware guardrails

  • Substantiation behind every benefit claim; no specific outcomes.
  • Prominent affiliate disclosure on every page; full disclosure page linked.
  • FTC endorsement: material-connection disclosure baked into every creator brief.
  • No before/after body imagery; no body shaming; no specific weight-loss promises.
  • 18+ age confirmation; under-18 blocked at intake.
  • GDPR / UK-GDPR / CCPA aligned; DSAR + opt-out endpoints.
  • PII columns encrypted with pgsodium; 90-day default retention for unconverted leads.
  • Geo gating where partner sells; no overselling out-of-region.
  • Per-vertical brand domain; warmed accounts; pixel separation.
  • Counsel review checkpoints: advertorial copy, every benefit claim, refund/cancel copy, every creator brief.

Top risks

  • FTC scrutiny on claims → counsel-reviewed copy + substantiation + no specific outcomes.
  • Refund / chargeback spikes → clear pre-purchase, cancel-anytime, partner refund SLA.
  • Creative fatigue → creator pipeline + bottom-30% rotation weekly.
  • Subscription churn → retention spine + adherence push + cycle review.
  • Partner payout shaving → server postbacks + monthly reconciliation.
  • Allergen / interaction risk → risk filter at quiz + safety disclaimer on results.
